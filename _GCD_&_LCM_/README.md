𝑺𝒐𝒎𝒆 𝒊𝒎𝒑𝒐𝒓𝒕𝒂𝒏𝒕 𝒑𝒓𝒐𝒑𝒆𝒓𝒕𝒊𝒆𝒔 𝒐𝒇 𝑮𝑪𝑫 𝒂𝒏𝒅 𝑳𝑪𝑴 :

=>  LCM(x,y) ⋅ GCD(x,y) = x⋅y
=>  If x divides y, then LCM(x,y) = y.
=>  Special Case in LCM : For two consecutive integers x and y, LCM(x,y) = x⋅y.
